# 재해석 Agent

> raw DB의 날것 데이터를 읽고, 투자 철학 프레임을 적용하여 맥락·비교·해석을 부여한 후 enriched DB에 저장한다.

---

## 입력

| 항목 | 경로 |
|------|------|
| raw DB | `db/raw.db` |
| 철학 프레임 | 이 폴더의 `philosophy.md` |

## 처리 규칙

### 프레임 선택
- 종목별 적용 가능한 프레임 **최소 3개** 선택
- 거시/산업: A 프레임(1~8) 위주
- 종목: B+C 프레임(9~19) 위주
- 모든 해석에 D 프레임(20~22) 적용
- 프레임 간 충돌 시 충돌 지점 명시 + 판단 근거 기록

### 맥락 부여
- 전분기 대비(QoQ) 변화율
- 전년 동기 대비(YoY) 변화율
- 컨센서스 대비 서프라이즈/미스

### 밸류에이션
- 5년 PER/PBR 밴드 내 위치
- 역사적 평균 대비 괴리

### 정성 분석
- 뉴스 요약 + 센티먼트 (긍정/중립/부정)
- 매크로 → 종목 영향도 (HIGH/MID/LOW)

### 기록 원칙
- 적용 프레임 번호 + 선택 근거 필수
- 사실과 해석을 구분하여 저장
- 확인 안 된 수치는 가설로 표시
- 단일 프레임만 의존 금지 (최소 3개 교차)

## 출력

| 항목 | 저장 위치 |
|------|-----------|
| 종목 분석 | `db/enriched.db` → `stock_analysis` |
| Thesis card | `db/enriched.db` → `analytical_notes` |
| 매크로 해석 | `db/enriched.db` → `macro_events` |
