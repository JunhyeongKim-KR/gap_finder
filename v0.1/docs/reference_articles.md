## CEO 태깅 가이드

각 글을 읽고 아래처럼 태깅해 주세요:

```
✅ 좋다 — 이유 한줄 (예: "논리 전개 구조가 우리 스타일")
❌ 아니다 — 이유 한줄 (예: "너무 학술적, 대중성 없음")  
⭐ 벤치마크 — 이 글 수준으로 만들고 싶다
```

이 태깅이 쌓이면 Reference DB + Feedback Loop의 초기 학습 데이터가 됩니다.
