## 현황 문서

| 문서 | 설명 | 언제 보나 |
|---|---|---|
| [progress.md](progress.md) | 전체 진행 상황, 완료/진행중/다음 할 일 | "지금 어디까지 왔어?" |
| [coverage.md](coverage.md) | 커버 중인 종목 목록, 판단, 분석글 링크 | "어떤 종목 다루고 있어?" |
| [changelog.md](changelog.md) | 날짜별 변경 이력 | "최근에 뭐가 바뀌었어?" |
