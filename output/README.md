# output — 종목 분석글

이 폴더에는 완성된 **종목 분석 글**이 저장됩니다.

## 파일 규칙

```
TICKER_종목명_YYYYMMDD.md
예: 005930_삼성전자_20260401.md
```

## 현재 발행된 글

| 파일 | 종목 | 발행일 |
|---|---|---|
| [005930_삼성전자_20260401.md](005930_삼성전자_20260401.md) | 삼성전자 | 2026-04-01 |

## 글 생성 방법

Claude에게 요청하면 됩니다:
```
"삼성전자 분석 글 만들어줘"
"SK하이닉스 역발상 관점으로 분석해줘"
```

## 글 검수 흐름

1. Claude가 초안 생성 → 이 폴더에 저장
2. CEO가 읽고 피드백 → `board/reviews/`에 리뷰 남김
3. 수정 반영 후 최종 확정
4. 플랫폼(티스토리/네이버)에 발행
