# 플랫폼 단계별 확장 전략

## 결정 내용

### Phase 1: 한국 중심 (초기)
- **네이버 블로그** + **티스토리** 동시 운영
- 네이버: 한국 검색 유입 + 초보자 타겟
- 티스토리: 구글 SEO + 애드센스 + 장문 원문 홈베이스
- 한국어 콘텐츠만 생산

### Phase 2: 영어권 확장 (트래픽 검증 후)
- 조건: 한국 채널에서 트래픽 증가 추세 확인된 이후
- **워드프레스 기반** 영문 사이트 배포
- 영어 콘텐츠는 기존 한국어 분석을 번역·재가공 또는 US 종목 영문 신규 작성
- 글로벌 SEO + 영어권 광고 단가 활용

## 근거
- 초기에 두 시장 동시 진출은 리소스 분산 → 한국 먼저 검증
- 워드프레스는 영문 SEO·플러그인 생태계·수익화에 최적
- 한국어 콘텐츠 DB가 쌓이면 영문 전환 비용이 낮아짐
