# 나스닥 관세 칼럼 피드백 — 핵심 논점 수정 요청

## 대상 글
- `output/NASDAQ_트럼프관세전쟁_20260406.md`

## 문제 진단

| 항목 | 평가 |
|------|------|
| 문장력 | 중상 |
| 시장 독해력 | 하 |
| 핵심 원인 | "주장 부족"이 아니라 **"주장 포인트가 낡았다"** |

### 구체적 문제

1. **관세의 직접 효과에 과도 집중, 금리 경로 누락**
   - 초안은 2024~2025식 '관세 = 공급망/실적 직접 노출' 프레임
   - 2026년 4월 시점 시장이 실제로 가격에 반영하는 **금리·인플레·가이던스 리스크 축**을 전면에 못 세움

2. **시장 해석 미스: transmission mechanism을 못 잡음**
   - 시장이 관세를 보는 1차 경로:
     ```
     관세 → 수입물가 상방 압력
     → 물가/기대인플레 상방 → Fed 완화 지연 또는 긴축 우려
     → 할인율 상방 → 장기 duration 자산인 성장주/빅테크 멀티플 압박
     ```
   - "빅테크가 관세에 직접 맞느냐"보다 **"관세가 금리 기대를 어떻게 흔드느냐"**가 먼저

3. **최신성 미흡**
   - 최근 소비자 1년 기대인플레이션 상승
   - Fed 인사들 인플레이션 안일론 경계 톤
   - 3월 고용지표 이후 금리 인하 기대 후퇴
   - 월가/Fed 주변 해석: "지정학/에너지/무역 불확실성 → 인플레 상방 → 인하 지연"

## 글을 살리려면

### 제목 방향
- "트럼프의 관세 총알, 나스닥이 맞은 건 실적이 아니라 할인율이다"
- "빅테크를 누르는 건 관세가 아니라 '인하 지연 공포'"
- "관세전쟁보다 무서운 건 금리의 재상승이다"

### 본문 논리 재배열

1. **시장은 왜 관세를 무서워하나**
   - 실적 직접 타격보다 기대인플레·금리 경로 훼손 때문

2. **왜 빅테크가 특히 아프나**
   - 장기 성장주일수록 할인율 변화에 멀티플 민감

3. **그런데 모든 빅테크가 같은가**
   - AAPL/NVDA: 공급망·규제 직접 노출
   - MSFT/GOOGL/META: 직접 노출보다 할인율/광고 경기 민감

4. **뭘 보면 되나**
   - CPI/PCE 기대
   - 10Y 실질금리
   - Fed 톤
   - 실적 시즌 가이던스에서 tariff보다 demand/FX/rate 언급 강도

## 향후 글쓰기 원칙 반영 사항

투자 글의 3대 필수 질문:
1. 지금 시장이 **무엇을 가격에 반영**하는지
2. 그 메커니즘이 **어떤 자산군에 어떻게 연결**되는지
3. 내 주장이 틀릴 경우 **무엇이 반증**인지
