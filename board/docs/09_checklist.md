# 9. 실행 체크리스트

## A. 종목 글 발행 전

- [ ] 티커/종목명/시장 구분 정확
- [ ] 최근 실적 발표 반영 확인
- [ ] 숫자 출처가 공식 자료(IR/공시/실적발표) — 출처 없는 숫자 본문 사용 금지
- [ ] 사업모델 설명이 초보자도 이해 가능
- [ ] 현재 시장 기대를 명시
- [ ] 투자 포인트가 "시장 기대와의 괴리"를 중심으로 전개
- [ ] 목표주가에 시나리오별 가정 명시
- [ ] 철회조건이 '가격 손절선'이 아니라 '가설 붕괴 조건'으로 정의
- [ ] 확인되지 않은 수치에 불확실성 표현 포함
- [ ] 사실 / 해석 / 추론 구분
- [ ] 초보 3줄 요약이 실제로 이해 가능한 수준
- [ ] 면책 문구 포함
- [ ] DB에 해당 종목 정보 모두 입력
- [ ] macro_exposures 태그 설정

## B. 매크로 이슈 발생 시

- [ ] macro_events 테이블에 이벤트 등록
- [ ] affected_tags와 매칭되는 종목 리스트 추출
- [ ] HIGH sensitivity 종목 우선 검토
- [ ] 투자 가설에 영향 있는지 판단
  - 영향 없음 → 글 상단 업데이트 노트만 추가
  - 영향 있음 → 목표주가/판단 재검토 후 본문 수정
- [ ] 업데이트 완료 후 update_log에 기록
- [ ] 수정된 글 멀티포스팅 재업로드
- [ ] action_taken = true 처리

## C. 플랫폼별 업로드 전

### 공통
- [ ] 제목에 핵심 키워드 포함 (종목명 + 전망/분석/목표주가)
- [ ] 본문 내 외부 링크 정상 작동
- [ ] 이미지/표 렌더링 확인

### 티스토리
- [ ] HTML/마크다운 포맷 정상 변환
- [ ] 카테고리 설정
- [ ] 태그 5~10개 설정
- [ ] 대표 이미지(썸네일) 설정

### 네이버 블로그
- [ ] 제목 네이버 SEO 최적화 (검색 키워드 앞배치)
