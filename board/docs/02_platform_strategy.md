# 2. 플랫폼 전략

## 확정 결정 (2026-04-01)

> 결정문서: `board/decisions/20260401_플랫폼_단계별_확장전략.md`

**한국 먼저 → 검증 후 영어권 확장** 전략이 확정되었다.

| 단계 | 플랫폼 | 설명 |
|------|--------|------|
| **Phase 1 (초기)** | 네이버 블로그 + 티스토리 | 한국어 콘텐츠만 생산. 네이버=검색 유입, 티스토리=구글 SEO+애드센스 홈베이스 |
| **Phase 2 (트래픽 검증 후)** | 워드프레스 (영문) | 한국 채널 트래픽 증가 추세 확인 이후. 기존 한국어 분석 번역·재가공 + US 종목 영문 신규 |

- 초기에 두 시장 동시 진출은 리소스 분산 → 한국 먼저 검증
- 영어권 확장은 Phase 3(5~6개월차) 이후 검토
- **Substack → WordPress로 변경**: 영문 SEO·플러그인 생태계·수익화에 워드프레스가 최적

## 추천 구조

- **본진: 티스토리** — 구글 SEO + 애드센스 + 자동화 연동 최적. 장문 분석글의 홈베이스.

- **보조 2: 워드프레스 (영문)** — ~~서브스택~~ → 워드프레스로 변경 (2026-04-01 결정). 영어권 확장 시 배포. 트래픽 검증 후 진행.
