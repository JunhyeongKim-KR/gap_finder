# Board — CEO·CTO 소통 공간

이 폴더는 CEO와 CTO가 서로 요청하고, 결정하고, 피드백하는 공간입니다.
직접 파일을 만들 필요 없이, Claude에게 말하면 알아서 정리해줍니다.

## 사용법

### 요청하기
```
"요청사항 남겨줘 — SK하이닉스 분석 추가, 반도체 사이클 관점으로"
```
→ `requests/` 폴더에 요청서가 생성됩니다.

### 결정 기록하기
```
"결정사항 — 이번 달은 반도체·2차전지 섹터 집중"
```
→ `decisions/` 폴더에 결정서가 생성됩니다.

### 리뷰 남기기
```
"삼성전자 글 리뷰 — 목표주가 근거 약함, EPS 기반 보완 필요"
```
→ `reviews/` 폴더에 피드백이 기록됩니다.

## 폴더 구조

```
board/
├── requests/     요청사항 (주로 CEO → CTO)
├── decisions/    확정된 결정 (CEO + CTO)
└── reviews/      글 리뷰·피드백 (주로 CEO)
```

## 상태 표시

각 문서 상단에 상태가 표시됩니다:
- **대기** — 아직 처리 안 됨
- **진행중** — 작업 중
- **완료** — 처리 완료
- **보류** — 나중에 처리
