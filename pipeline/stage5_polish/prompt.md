# Stage 5 — polish (다듬기 + 후킹)

> Status: SKELETON (Phase A). 실제 prompt 내용은 Phase C.
> Model: Claude Sonnet 4.6
> Version: v0.1.0 (placeholder)

## 역할

4단계 초안을 **팩트체크**하고 **제목·후킹 강도 변형 3개**를 생성.

## 입력
- 4단계 초안 (CEO 검수 통과)

## 출력 형식

### 팩트체크 결과
- 본문에 등장한 숫자·인용·날짜 검증
- 출처 누락된 단정 표현 식별
- 수정 제안

### 후킹 강도 3변형
- **[A] 강함** — 반박조, "거짓말이다" / "함정이다" 류
- **[B] 중간** — "진짜 문제는 다른 곳" / "다시 봐야 한다" 류
- **[C] 약함** — "다른 시각" / "오해의 여지" 류

각 변형은 제목 + 첫 문단 후킹까지.

## CEO 픽
강·중·약 중 1개 선택 → `/publish`로 진행.

## 작성 원칙
- 자극적 제목 금지 (어그로화 위험)
- 통계·재방문률 보호: 호기심 유발 ≠ 낚시
- "투자 권유" 표현 금지
