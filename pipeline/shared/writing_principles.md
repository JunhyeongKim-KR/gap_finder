# 글쓰기 원칙

> 모든 5단계 prompt가 참조하는 공통 자산.

## 글의 기본 문법

```
[시장 컨센서스] → [해석의 맹점] → [실제 메커니즘] → [진짜 수혜/피해] → [체크포인트]
```

## 절대 원칙 (위반 시 reject)

1. **확인되지 않은 수치는 절대 단정 금지**
2. **공식 자료(IR/공시/실적발표/공식 통계) 확인 시에만 숫자 제시**
3. **불확실하면 명시적으로 표현** — "확인되지 않으나…", "가설적 추론 전제"
4. **사실·해석·추론을 반드시 구분** — 사실 진술과 해석을 섞지 않음
5. **목표주가는 시나리오와 가정을 명시** — 단일 숫자가 아닌 베이스/불/베어 구조
6. **철회조건은 '가격'이 아니라 '가설 붕괴 조건'** — "X% 손실 시 손절"이 아니라 "점유율 하락 구조적 확인 시"

## 톤

- 자극적·낚시성 표현 금지
- "투자 권유" 표현 금지 — 판단 근거 중심
- 초보자도 읽을 수 있는 사업모델 설명 (단, 깊이는 유지)
- 한 줄 결론 명확히

## 표준 섹션 (4단계 draft에서 사용)

| 섹션 | 내용 |
|------|------|
| 제목 | [종목명] 시장이 이 회사를 오해하는 이유 / 지금 싸다·비싸다를 어떻게 봐야 하나 |
| 초보자 3줄 요약 | ① 이 회사가 뭘로 돈 버는지 ② 시장이 지금 뭘 걱정하는지 ③ 내 판단 |
| 한 줄 결론 | 예: '단기 악재 과대반영 구간 — 6~12개월 재평가 가능성 높음' |
| 기업 개요 | 사업 구조 / 핵심 제품 / 고객 / 지역 노출 |
| 비즈니스모델 | 매출 발생 구조 / 이익률 개선 조건 / 진입 장벽 |
| 산업 및 경쟁구도 | 시장 구조 / 경쟁사 포지션 / 가격결정력 |
| 실적 해석 | 최근 실적 핵심 / 컨센서스 대비 / 일회성 vs 구조적 변화 |
| 현재 주가 판단 | 시장이 반영한 기대 / 내가 보는 괴리 |
| 목표주가 | 베이스·불·베어 시나리오 / 가정 / 멀티플 / 범위 |
| 투자 철회 조건 | 가설 붕괴 조건 (가격 손절 ❌) |
| 상승/하락 시나리오 | 시나리오별 트리거 |
| 최종 판단 | 명확한 스탠스, 판단 근거 중심 |
| 마무리 문장 | 핵심 메시지 |

## 발행 전 체크리스트

- [ ] 티커/종목명/시장 구분 정확
- [ ] 최근 실적 발표 반영
- [ ] 모든 숫자에 출처
- [ ] 사업모델 설명 명료
- [ ] 현재 시장 기대 명시
- [ ] 목표주가 가정 명시
- [ ] 철회조건이 가설 붕괴 조건
- [ ] 한 줄 결론 모호하지 않음
