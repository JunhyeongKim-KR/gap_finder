# 11. 기존 PDF 전략 비교 분석

기존 PDF("투자리서치_사업청사진_로드맵")와 이 시스템 전략의 비교.

## PDF가 더 나은 점

| 항목 | 설명 |
|---|---|
| **2인 부업 + 저자본 전제** | 인력 구조와 비용 구조가 처음부터 명시 |
| **3-Tier 발행 구조** | 신규 심화 / 업데이트 / 재가공으로 일 10건 달성 가능한 구조 |
| **목표주가 방법론 4유형** | 컨센서스참고형 / 절대가치평가형 / 상대가치평가형 / 이벤트리버전형 |
| **투자 아이디어 taxonomy 7가지** | 역발상의 구체적 유형 분류 |
| **자동화 OK/금지 경계선** | 무엇을 자동화하고 무엇을 사람이 할지 명확 |
| **P&L 프로젝션** | PV당 수익 가정 + 단계별 시나리오 |
| **핵심 원칙 5조** | 운영 규율이 원칙으로 고정 |
| **n8n + Baserow 툴 스택** | 자동화 엔진까지 확정 |

## 이 시스템이 더 나은 점

| 항목 | 설명 |
|---|---|
| **DB 필드 설계** | 60+ 필드 + 보조 테이블 3개. 매크로 연동 자동화에 필수 |
| **매크로-종목 연동 로직** | macro_exposures + sensitivity + 우선순위 4단계 |
| **종목 선정 기준** | MECE 9가지 + Value Trap 필터 |
| **운영 리스크 매트릭스** | 6가지 리스크 × 구체적 통제 방안 |
| **글 템플릿 완성도** | 업데이트 이력, 확신도, 보유기간, 면책 문구 등 |
| **플랫폼 평가 6기준 매트릭스** | 체계적 비교 |
| **트랙레코드 투명 공개 체계** | verdict_history, 월간 리뷰, 틀린 이유 분석 콘텐츠화 |

## 일치하는 부분

- 역발상 + 중기매매 철학
- 티스토리 본진 + 네이버 보조
- Substack은 후순위
- 글 템플릿 기본 구조 (~90%)
- 발행 전 체크리스트 (~85%)
- "사실/해석/추론 구분" 원칙
- 피해야 할 종목 유형
- 자동화는 검증 후 도입
