# 5. 매크로 이슈 업데이트 체계

## 매크로 이슈 분류

| 카테고리 | 구체적 이슈 | 업데이트 긴급도 |
|---|---|---|
| **금리** | 기준금리 변동, 국채수익률 급변, 금리 전망 변경 | 24시간 내 |
| **환율** | 원/달러 급변(±3%), 엔/달러 급변, 달러인덱스 방향 전환 | 24시간 내 |
| **원자재** | 유가 ±10%, 구리·리튬 등 핵심 원자재 급변 | 48시간 내 |
| **산업 사이클** | 반도체 ASP 전환, 재고 사이클 변동, 수주 트렌드 변화 | 48시간 내 |
| **규제/정책** | 관세 변경, 보조금 정책, 산업 규제 변화 | 48시간 내 |
| **지정학** | 전쟁/갈등 확대, 공급망 차단, 제재 | 24시간 내 |
| **실적 시즌** | 주요 기업 실적 발표 (해당 종목 + 경쟁사/고객사) | 72시간 내 |
| **센티먼트** | VIX 급등(30+), 시장 패닉/유포리아, 섹터 로테이션 | 48시간 내 |

## 우선순위 로직

이슈 발생 시 업데이트 순서:

```
1순위: macro_sensitivity에서 해당 요인이 HIGH인 종목
2순위: 해당 요인이 MID인 종목 중 verdict가 BUY인 종목
3순위: 나머지 MID 종목
4순위: LOW 종목은 월간 정기 리뷰 시 반영
```

## 업데이트 시 수정 항목

| 항목 | 수정 여부 | 조건 |
|---|---|---|
| 투자 포인트(thesis) | 가설 자체가 영향받을 때만 | 드물게 |
| 밸류에이션 수치 | 실적 추정 변경 시 | 분기별 |
| 목표주가 시나리오 | 가정이 변경될 때 | 이슈 규모에 따라 |
| 철회 조건 | 트리거 근접 또는 새 리스크 발생 시 | 필요 시 |
| 판단(verdict) | 가설 훼손 또는 대폭 강화 시 | 드물게 |
| 글 상단 업데이트 노트 | **매번** | 항상 |

## 업데이트 노트 형식

```
📌 [2026.03.28 업데이트] 미 기준금리 25bp 인하 결정에 따른 밸류에이션 재산정.
목표주가 기본 시나리오 ○○원 → ○○원 상향. 판단 유지: 매수.
```
